# Sistema de Ponto

Este é um sistema de controle de ponto desenvolvido em PHP.

**Status: Em Desenvolvimento**

Este projeto está atualmente em fase de desenvolvimento. Funcionalidades podem ser adicionadas, modificadas ou removidas sem aviso prévio.

## Instalação

1. Clone o repositório.
2. Configure o banco de dados usando o arquivo `database.sql`.
3. Execute o `setup_permissions.php` e outros scripts de configuração conforme necessário.

## Uso

Acesse `index.php` para começar.
